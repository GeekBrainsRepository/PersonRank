Программа запускается посредством запуска main метода в Application.java. Класс
PagesCreater предназначен для заполнения БД ссылками на страницы сайта.
Parser выполняет все манипуляции по поиску ключевых слов в тексте страницы.
Для загрузки и парсинга страниц использована библиотека Jsoup.
Маппинг БД - MyBatis.
Фреймворк - Spring.
Логгер - log4j.
Тестирование - junit.
