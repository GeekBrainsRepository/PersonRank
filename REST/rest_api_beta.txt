Вообщем примерный вид API будет таким:
Для статистик 


service.addr/statistic/getpersonlist
Возвращает список персон
Предварительный формат данных:
JSON , пары ключ:значение  (id:person_name)
Пример:
{"1":"First","2":"Second"}


service.addr/statistic/daily/{siteId}/{personId}/{dateStart}/{dateEnd}
возвращает статистику по реурсу site_id для персоны person_id за дату date
service.addr - адрес веб-сервиса
{site_id} - ID сайта из таблицы Sites
{person_id} - ID персоны из таблицы Persons
{date_start} - дата старта выборки статистики ?? в формате long
{date_end} - дата конца выборки статистики ?? в формате long

service.addr/statistic/common/{siteId}
возвращает общую статистику по ресурсу site_id
service.addr - адрес веб-сервиса
{site_id} - ID сайта из таблицы Sites
